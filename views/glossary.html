Glossary

Aristotle Library Apps
The Aristotle Library Apps project is based on Django and provides the web front-end to a Redis-based BIBFRAME Datastore. Source code is available at https://github.com/jermnelson/aristotle-library-apps
BIBFRAME Datastore
An open-source project, not affiliated with the Library of Congress, that has configuration, documentation, and custom scripting for a native BIBFRAME datastore implemented in Redis. Source code is available at https://github.com/jermnelson/BIBFRAME-Datastore.
Lean Manufacturing
A management and production philosophy, pioneered by Toyota, that strives to manufacture a product based on the demand of the customer with a focus on just-in-time delivery of inputs and a reduction of waste.
Lean Startup
The Lean Startup movement takes ideas and processes from Lean Manufacturing and applies these principles to start-up businesses while offering ideas and applications to established businesses, government, and non-profit organizations like libraries.
Memory Theatres
Memory Theatres, or alternatively known as Method of Loci or Memory palace is a memorization technique first used by the ancient Greek and Roman. This technique associates an item to remembered with a specific physical location. Scholars would create rich and descriptive environments both physically and in their minds for memorizating unrelated information.
Open Access
Open Access is a designation for professional and trade articles that are digital, free-to-access, and free of most copyright or licensing restrictions.
Open Source
Open Source refers to software and documentation that is licensed under a permissive license that usually allows for free copying and use of the source code as long as the derived changes are suitable attributed. Some open source license require any code contributions to be released under the same license (see GPL license) while other licenses are less restrictive for requiring code contributions back to the project.
Pull Platform
A platform made up of people and technology offering a wider range of loosely-coupled options for connecting library patrons and staff to resources managed by the library.
